## 1. 개요

### 1.1 목적
본 문서는 C++ CLI 기반 4칙 연산 계산기 소프트웨어의 소프트웨어 요구사항(SW Requirements)을 식별·정의하고, 상위 목표(고객 사양)와의 추적성을 확보한다.

### 1.2 범위
- 대상 소프트웨어: CLI 계산기 (`calculator`)
- 주요 기능: 정수/실수 4칙 연산(덧셈·뺄셈·곱셈·나눗셈), 예외 처리, CLI 입출력, 반복 루프, 종료 처리
- 실행 환경: Ubuntu 22.04 LTS, CLI(표준 입출력)

### 1.3 용어 정의

| 용어 | 정의 |
|------|------|
| 피연산자 | 연산에 사용되는 수 (실수형 `double` 기준) |
| 연산자 | `+`, `-`, `*`, `/` 중 하나 |
| 예외 | 0 나누기와 같이 정상적으로 처리할 수 없는 입력 상황 |
| 종료 명령 | 사용자가 `q` 또는 `quit`을 입력하여 프로그램을 정상 종료하는 명령 |

| 제목 | 나눗셈 연산 |
| 내용 | 두 개의 실수형 피연산자 `a`, `b`에 대해 `a / b`를 계산하고 결과를 반환·출력한다. `b == 0`인 경우는 SWE-REQ-0005에 따른다. |

### SWE-REQ-0005 — 0 나누기 예외 처리

| 항목 | 내용 |
|------|------|
| ID | SWE-REQ-0005 |
| 제목 | 0 나누기 예외 처리 |
| 내용 | 나눗셈 수행 시 제수가 0이면 오류 메시지(예: `"Error: Division by zero"`)를 표준 출력에 출력하고 연산을 중단한다. 프로그램은 비정상 종료되지 않으며 반복 루프를 유지한다. |
| 입력 | `a: double`, `b: double` (b == 0) |
| 출력 | 오류 메시지 문자열 (표준 출력) |
| 사전 조건 | `b == 0` |
| 사후 조건 | 프로그램 정상 동작 유지, exit code 변경 없음 |
| 예외 | 없음 (이 요구사항 자체가 예외 처리 명세) |
