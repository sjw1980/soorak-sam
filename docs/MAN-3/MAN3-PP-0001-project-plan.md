## 1. 프로젝트 개요

### 1.1 목적
C++ CLI 기반 4칙 연산 계산기 소프트웨어를 ASPICE PAM 3.1 SWE-1~SWE-6 프로세스에 따라 개발·검증한다.

### 1.2 범위
| 포함 | 제외 |

| SWE-3 상세 설계 | 배포 인프라 구성 |
| SWE-4 단위 검증 (Google Test) | |
| SWE-5 통합 테스트 | |
| SWE-6 적격성 테스트 | |

### 1.3 완료 기준
- 모든 SWE-REQ 요구사항이 SWE-QTC 테스트 케이스에 의해 검증됨
- 단위 커버리지(문장 100%, 분기 100%) 달성
- 모든 SWE-6 적격성 테스트 Pass
- 산출물 버전 Approved 상태
